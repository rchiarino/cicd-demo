## Introducción

Esta documentación fue creada como ejemplo sobre el uso de [GitHub Actions](https://github.com/features/actions) y la implementación de un pipeline de [CI/CD](https://github.com/resources/articles/devops/ci-cd) (Integración Continua/Despliegue Continuo). Este material tiene como objetivo demostrar cómo automatizar pruebas, compilaciones y despliegues de código utilizando GitHub Actions, facilitando así el desarrollo ágil y eficiente en proyectos de software.

A lo largo de toda la documentación, se incluyen enlaces a las fuentes que se utilizaron para estudiar y profundizar en el tema. Permitiendo a los lectores ampliar sus conocimientos y aplicar los conceptos en sus propios proyectos.
